getMetadata

Retrieve the metadata for this log from the object.

Declaration
public object getMetadata { get; }
Property Value
Type Description
System.Object

The metadata for this log. The object returned is essentially a JSON object. To deal with an object with an 'unknown' class (dynamic), the object is of type JObject (http://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Linq_JObject.htm). See the newtonsoft documentation for more details on how to use this object to get to all the data contained in it. Also see https://weblog.west-wind.com/posts/2012/Aug/30/Using-JSONNET-for-dynamic-JSON-parsing for more information about dealing with dynamic JSON parsing.
